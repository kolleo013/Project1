Название проекта: Personal Portfolio Website
Описание: Личный одностраничный сайт-визитка, разработанный для представления навыков frontend-разработки. Проект полностью адаптирован под мобильные устройства.
Стек технологий: HTML5, CSS3 (Flexbox/Grid), JavaScript (ES6+), Git
Функциональность:

· Адаптивный дизайн
· Интерактивное меню
· Секция портфолио с проектами
  Как запустить проект: Склонируйте репозиторий и откройте файл index.html в браузере.
